# Markdown

Markdown
タスクの説明文は、はじめるボタンの表示の際のタスクの説明文を定義するために利用します。
タスクの説明文は Markdown https://help.docbase.io/posts/13697 により修飾することができます。

> Markdownの他、HTMLを記述することができます。特にHTMLのブロック要素を記述する場合は、前後にスペースを入れてください。
